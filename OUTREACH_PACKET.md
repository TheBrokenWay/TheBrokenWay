## 2. 100-word company summary

Tillar Technologies is building governed systems for high-trust workflows. The
core design direction is simple: the model does work, but the model does not
govern itself. That governance remains outside the model in an independent
operating path. The current product family includes PredX for governed
scientific decision support, CageOS for bounded operational governance, and
CageOS Lite as a public proof surface. The public demo is intentionally small:
it demonstrates the control pattern without disclosing the private enforcement
mechanics.
